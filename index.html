About The Attendance System

This is a live RFID based Attendance System which would record the entry and exit times of IEEE members to IEEE Maker's Space. Each User's RFID Card is registered on the NodeMCU board with the help of RFID Card Reader. With the help of Google Firebase Realtime Database all entry and exit times are recorded which are then presented live on website
